# Justificativa Técnica

Foi adotada uma arquitetura baseada em microsserviços visando modularidade, escalabilidade e facilidade de manutenção.

Foi utilizada Arquitetura Limpa para separar responsabilidades.

A divisão em camadas reduz acoplamento e facilita testes.

O React foi escolhido para oferecer uma interface moderna e dinâmica.

Docker foi utilizado para padronizar o ambiente de execução.

MySQL foi escolhido pela consistência transacional e facilidade de integração.

JWT foi utilizado para autenticação stateless.
